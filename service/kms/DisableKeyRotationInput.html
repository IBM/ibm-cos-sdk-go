Structure Field Details

KeyId *string `min:"1" type:"string" required:"true"`

Identifies a symmetric customer master key (CMK). You cannot enable or disable automatic rotation of asymmetric CMKs (docs.aws.amazon.com/kms/latest/developerguide/symmetric-asymmetric.html#asymmetric-cmks), CMKs with imported key material (docs.aws.amazon.com/kms/latest/developerguide/importing-keys.html), or CMKs in a custom key store (docs.aws.amazon.com/kms/latest/developerguide/custom-key-store-overview.html).

Specify the key ID or the Amazon Resource Name (ARN) of the CMK.

For example:

  • Key ID: 1234abcd-12ab-34cd-56ef-1234567890ab

  • Key ARN: arn:aws:kms:us-east-2:111122223333:key/1234abcd-12ab-34cd-56ef-1234567890ab

To get the key ID and key ARN for a CMK, use ListKeys or DescribeKey.

KeyId is a required field
